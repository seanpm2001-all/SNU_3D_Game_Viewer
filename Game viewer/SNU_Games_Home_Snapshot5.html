Game prices

SNU Games is going to have an assortment of games. But what will the price be?

The creator of the game can choose a price for their game. They can also choose special attributes such as [OPEN SOURCE] which means you can modify the source of the game, [CLOSED SOURCE] which means that the game is locked down, and the source is a lot harder to edit.

Prices of games can be between [FREE] and [49.99] they cannot go any higher than that. If you are a developer and need ways of making more revenue, try creating a donation page, or add in special in-app purchases.

In-app purchases have set rules, so they cannot be abused. For one, the game has to be playable without spending a single penny on in-app purchases. So if you have to pay to progress, the game will be removed, as that is not how a game should be.

Support

SNU Games has a lot of support for older machines, and legacy software.

There is support for all games here. Games can be played both offline and online.

We support the following processor types:

* 4 bit (virtual machine/emulator only)

* 8 bit (virtual machine/emulator only)

* 16 bit (virtual machine/emulator only)

* 32 bit (virtual machine/emulator only)

* 64 bit

* 128 bit

* 256 bit (coming in the near future)

The following operating systems are supported:

Linux

Windows

MacOS

BSD
